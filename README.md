# Country_Wise_Insight_Of_Online_Retail_Based_On_Revenue

Retail Sales Analysis Dashboard - Power BI Project

Project Overview

This project is a comprehensive retail sales analysis dashboard created using Power BI. The dashboard provides insights into sales trends, customer segmentation, and regional demand across multiple countries (excluding the UK), helping businesses make informed strategic decisions.

Features

Monthly sales and revenue trend analysis using line charts.
Country-wise revenue and quantity sold with detailed visual mapping.
Identification of top 10 customers by revenue to target high-value clients.
Regional demand visualization with an interactive map and color-coded sales quantity.

Tools Used

Power BI for data visualization and dashboard creation.
Excel for initial data cleaning and transformation.

Data Cleaning and Preparation

The data went through thorough cleaning and preprocessing to ensure accuracy. This includes:
Removal of duplicate and invalid entries.
Exclusion of UK sales data as per project requirements.
Data type corrections and aggregations for smooth visualization.

How to Use

Open the Power BI .pbix file included in this repository using Power BI Desktop to explore the interactive dashboard and insights.

Learning Outcomes

Developed proficiency in real-world data visualization using Power BI.
Enhanced understanding of sales data patterns and customer segmentation for business insights.
